# 网站设计系统与列表模板规范

适用范围：cms 站点通用页面，尤其是列表页与图文列表页。配合[页面样式规范](./页面样式规范.md)落地。

## 一、信息架构与页面类型

- 全局：页头、主导航、面包屑、主内容、侧栏（可选）、页脚
- 页面类型：
  - 纯列表页：标题 + 元信息 + 摘要
  - 图文列表页：缩略图 + 标题 + 元信息 + 摘要
  - 详情页：标题、时间来源、正文、附件、上一篇/下一篇
  - 检索页：搜索、筛选、结果列表、分页

## 二、布局与响应式

- 容器宽度：1200px 居中，移动端全宽
- 栅格：12 列，列间距 24，外侧 16
- 断点：
  - ≥1200：桌面
  - 768–1199：平板
  - <768：手机

```css
.container { max-width: 1200px; margin: 0 auto; padding: 0 16px; }
.grid { display: grid; grid-template-columns: repeat(12, 1fr); grid-gap: 24px; }
@media (max-width: 1199px) { .grid { grid-gap: 20px; } }
@media (max-width: 767px) { .grid { grid-template-columns: 1fr; grid-gap: 16px; } }
```

## 三、基础令牌与工具类

沿用变量：颜色、字号、间距参见页面样式规范。补充工具类：

```css
.mt-2 { margin-top: 8px; } .mt-3 { margin-top: 12px; } .mt-4 { margin-top: 16px; }
.mb-2 { margin-bottom: 8px; } .mb-3 { margin-bottom: 12px; } .mb-4 { margin-bottom: 16px; }
.text-muted { color: var(--color-text-secondary); }
.card { background: var(--color-bg); border: 1px solid var(--color-border); border-radius: var(--radius-md); }
.media { display: flex; gap: 16px; align-items: flex-start; }
@media (max-width: 767px) { .media { flex-direction: column; } }
```

## 四、列表组件（纯文本）

结构：

```html
<ul class="list">
  <li class="list-item">
    <a class="list-title" href="#">标题</a>
    <div class="list-meta">2026-03-06 来源</div>
    <p class="list-summary">摘要内容</p>
  </li>
</ul>
```

样式：

```css
.list { list-style: none; padding: 0; margin: 0; }
.list-item { padding: 16px 0; border-bottom: 1px solid var(--color-border); }
.list-title { font-size: 18px; line-height: 1.5; color: var(--color-text-strong); }
.list-title:hover { color: var(--color-primary); }
.list-meta { margin-top: 8px; font-size: 14px; color: var(--color-text-secondary); }
.list-summary { margin-top: 8px; font-size: 16px; line-height: 1.6; color: var(--color-text); }
```

## 五、图文列表组件（媒体对象/卡片）

结构：

```html
<ul class="media-list">
  <li class="media card">
    <a class="media-thumb" href="#"><img src="thumb.jpg" alt=""></a>
    <div class="media-body">
      <a class="media-title" href="#">标题</a>
      <div class="media-meta">2026-03-06 类别</div>
      <p class="media-summary">摘要内容</p>
    </div>
  </li>
</ul>
```

样式：

```css
.media-list { list-style: none; padding: 0; margin: 0; }
.media { padding: 16px; }
.media-thumb { flex: 0 0 220px; display: block; overflow: hidden; border-radius: var(--radius-sm); }
.media-thumb img { width: 100%; height: 140px; object-fit: cover; display: block; }
.media-body { flex: 1; }
.media-title { font-size: 18px; line-height: 1.5; color: var(--color-text-strong); }
.media-title:hover { color: var(--color-primary); }
.media-meta { margin-top: 8px; font-size: 14px; color: var(--color-text-secondary); }
.media-summary { margin-top: 8px; font-size: 16px; line-height: 1.6; color: var(--color-text); }
@media (max-width: 767px) { .media-thumb { width: 100%; } .media-thumb img { height: 180px; } }
```

## 六、分页组件

结构：

```html
<nav class="pagination" aria-label="分页导航">
  <a class="page prev" href="#">上一页</a>
  <a class="page" href="#">1</a>
  <a class="page active" href="#" aria-current="page">2</a>
  <a class="page" href="#">3</a>
  <a class="page next" href="#">下一页</a>
</nav>
```

样式：

```css
.pagination { display: flex; gap: 8px; flex-wrap: wrap; margin-top: 16px; }
.page { min-width: 36px; height: 36px; padding: 0 12px; border: 1px solid var(--color-border); border-radius: var(--radius-sm); display: inline-flex; align-items: center; justify-content: center; color: var(--color-text); background: var(--color-bg); }
.page:hover { border-color: var(--color-primary); color: var(--color-primary); }
.page.active { border-color: var(--color-primary); background: rgba(201,25,25,.06); color: var(--color-primary); }
```

## 七、筛选与搜索

结构：

```html
<form class="filters">
  <label>关键词<input type="text" placeholder="请输入关键词"></label>
  <select><option>全部类别</option></select>
  <button type="submit">查询</button>
</form>
```

样式：

```css
.filters { display: flex; gap: 12px; flex-wrap: wrap; margin-bottom: 16px; }
.filters label { display: inline-flex; align-items: center; gap: 8px; }
.filters input[type="text"] { width: 280px; }
@media (max-width: 767px) { .filters input[type="text"] { width: 100%; } }
```

## 八、面包屑与导航

结构：

```html
<nav class="breadcrumb" aria-label="面包屑">
  <a href="/">首页</a>
  <span class="sep">/</span>
  <a href="#">栏目</a>
  <span class="sep">/</span>
  <span aria-current="page">当前页</span>
</nav>
```

样式：

```css
.breadcrumb { display: flex; flex-wrap: wrap; gap: 8px; margin: 16px 0; font-size: 14px; color: var(--color-text-secondary); }
.breadcrumb a { color: var(--color-text-secondary); }
.breadcrumb a:hover { color: var(--color-primary); }
.breadcrumb .sep { color: var(--color-border); }
```

## 九、详情页规范

- 标题使用 h1，字号 28，行高 1.4，间距上 24 下 16
- 元信息 14px 次要色
- 正文 p 16px 行高 1.7
- 图片最大宽度 100%，带标题文字可居中显示
- 附件列表采用列表样式与图标

```css
.article-title { font-size: 28px; line-height: 1.4; color: var(--color-text-strong); margin: 24px 0 16px; }
.article-meta { font-size: 14px; color: var(--color-text-secondary); margin-bottom: 16px; }
.article-content img { max-width: 100%; height: auto; display: block; margin: 12px auto; }
```

## 十、命名与可访问性

- 命名：BEM，如 list、list__item、list__title；媒体对象 media、media__thumb、media__body
- 可访问性：链接焦点可见，分页 aria 标注，图片 alt 文案，按钮语义化

## 十一、落地步骤

- 将 CSS 变量与基础样式写入全局样式文件
- 各列表页复用上述结构与类名
- 白标主题调整仅变更 CSS 变量
- 新需求新增组件前先复用模板，无法满足再扩展
