# 页面样式规范（cms.zytzb.gov.cn）

适用范围：页面 `https://cms.zytzb.gov.cn/eportal/ui?pageId=343e729521c2454c84c42e9bb201b728`

目标：统一字体、字号、标签语义、间距与颜色，确保可读性、一致性与可维护性。

## 一、字体与字重

- 字体家族：system-ui, -apple-system, Segoe UI, Roboto, Noto Sans, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Source Han Sans SC", "Noto Sans CJK SC", Arial, sans-serif
- 字重使用：标题 600–700；正文 400；强调 500–600；避免使用 800 以上大字重
- 字间距：中文正常值；英文字母与数字保持默认

## 二、字号与行高

| 用途 | 标签 | 字号(px) | 行高 | 说明 |
|---|---|---:|---:|---|
| 页面主标题 | h1 | 28 | 1.4 | 页面唯一，避免堆叠 |
| 章节标题 | h2 | 24 | 1.4 | 模块一级标题 |
| 小节标题 | h3 | 20 | 1.45 | 模块二级标题 |
| 正文 | p | 16 | 1.6 | 默认正文 |
| 辅助说明 | small, .text-muted | 14 | 1.6 | 备注、脚注 |
| 按钮文字 | button | 16 | 1.0 | 水平留白保证可点区 |
| 表单标签 | label | 16 | 1.4 | 与输入框垂直对齐 |
| 表格正文 | table td | 14–16 | 1.5 | 视密度可选 |

## 三、标签与语义

- 标题层级：页面仅一个 h1；模块标题用 h2；子模块用 h3；避免跳级
- 段落与列表：正文用 p；成组信息用 ul/ol + li；不使用 div 代替语义标签
- 链接：a，默认带下划线；悬停变色；焦点可见
- 强调：strong 或 em 控制语义，不用纯样式标签
- 图片：img 必须有 alt 文案
- 表单：label 关联 input；必填项标识放在 label 内部

## 四、颜色规范

| 名称 | 变量 | HEX | 用途 |
|---|---|---|---|
| 主色 | --color-primary | #C91919 | 品牌主按钮、链接强调 |
| 深主色 | --color-primary-600 | #A51414 | 悬停、按压 |
| 链接 | --color-link | #0B5FFF | 文本链接 |
| 成功 | --color-success | #2E7D32 | 成功状态 |
| 警告 | --color-warning | #ED6C02 | 警告状态 |
| 错误 | --color-danger | #D32F2F | 错误状态 |
| 标题文本 | --color-text-strong | #1F2329 | 主标题 |
| 正文文本 | --color-text | #2C2F36 | 正文 |
| 次要文本 | --color-text-secondary | #5F6B7A | 说明文字 |
| 边框 | --color-border | #E5E7EB | 分隔与边框 |
| 背景 | --color-bg | #FFFFFF | 主背景 |
| 灰背景 | --color-bg-subtle | #F7F8FA | 模块底色 |

对比度：正文文本与背景对比度≥4.5:1，标题≥3:1。

## 五、间距与圆角

- 间距刻度：4, 8, 12, 16, 20, 24, 32(px)
- 模块外边距：24–32
- 模块内边距：16–24
- 文本段落间距：12–16
- 列表项目间距：8
- 表单控件垂直间距：12–16
- 圆角：按钮与输入框 4；卡片 6–8

## 六、按钮与表单

| 类别 | 高度(px) | 字号 | 内边距（左右） | 圆角 | 说明 |
|---|---:|---:|---:|---:|---|
| 大按钮 | 40 | 16 | 16 | 4 | 主操作 |
| 常规按钮 | 36 | 16 | 14 | 4 | 常用 |
| 小按钮 | 32 | 14 | 12 | 4 | 次要 |

表单输入框高度：36；字号 16；左右内边距 12；边框 1px，颜色 --color-border。焦点边框使用主色深色。

## 七、状态与交互

- 悬停：背景或边框加深 10–15%，文字不变或略深
- 按压：背景加深 15–20%，移除阴影
- 禁用：不透明度 60%，禁用指针事件
- 焦点可见：采用 outline 或边框高亮，确保键盘可达

## 八、实现参考（CSS 变量与基础样式）

```css
:root {
  --color-primary: #C91919;
  --color-primary-600: #A51414;
  --color-link: #0B5FFF;
  --color-success: #2E7D32;
  --color-warning: #ED6C02;
  --color-danger: #D32F2F;
  --color-text-strong: #1F2329;
  --color-text: #2C2F36;
  --color-text-secondary: #5F6B7A;
  --color-border: #E5E7EB;
  --color-bg: #FFFFFF;
  --color-bg-subtle: #F7F8FA;

  --radius-sm: 4px;
  --radius-md: 6px;
  --space-1: 4px;
  --space-2: 8px;
  --space-3: 12px;
  --space-4: 16px;
  --space-5: 20px;
  --space-6: 24px;
  --space-8: 32px;
}

html, body {
  font-family: system-ui, -apple-system, Segoe UI, Roboto, Noto Sans, "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Source Han Sans SC", "Noto Sans CJK SC", Arial, sans-serif;
  color: var(--color-text);
  background: var(--color-bg);
}

h1 { font-size: 28px; line-height: 1.4; color: var(--color-text-strong); margin: var(--space-6) 0 var(--space-4); }
h2 { font-size: 24px; line-height: 1.4; color: var(--color-text-strong); margin: var(--space-5) 0 var(--space-3); }
h3 { font-size: 20px; line-height: 1.45; color: var(--color-text-strong); margin: var(--space-4) 0 var(--space-2); }
p  { font-size: 16px; line-height: 1.6; margin: var(--space-3) 0; }
small, .text-muted { font-size: 14px; color: var(--color-text-secondary); }

a { color: var(--color-link); text-decoration: underline; }
a:hover { color: var(--color-primary); }
a:focus { outline: 2px solid var(--color-primary-600); outline-offset: 2px; }

button {
  height: 36px;
  padding: 0 14px;
  font-size: 16px;
  color: #fff;
  background: var(--color-primary);
  border: 1px solid var(--color-primary);
  border-radius: var(--radius-sm);
}
button:hover { background: var(--color-primary-600); border-color: var(--color-primary-600); }
button:disabled { opacity: .6; pointer-events: none; }

input, select, textarea {
  height: 36px;
  padding: 0 12px;
  font-size: 16px;
  border: 1px solid var(--color-border);
  border-radius: var(--radius-sm);
}
input:focus, select:focus, textarea:focus {
  outline: none;
  border-color: var(--color-primary-600);
  box-shadow: 0 0 0 2px rgba(201,25,25,.15);
}
```

## 九、可访问性

- 焦点状态必须可见
- 图片与图标提供替代文本
- 表单控件具备 label 与 aria 描述
- 颜色对比符合 WCAG AA

## 十、交付与维护

- 优先采用上述 CSS 变量落地
- 新模块新增颜色或字号时，从刻度中选择，避免自定义散点
- 变更需评估对比度与可读性
